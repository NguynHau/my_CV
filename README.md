# CV – Typst

This repository contains my personal CV written in Typst and exported as a PDF.

## 📄 Preview

👉 [Download CV (PDF)](./CV_NguyenTanHau.pdf)

## 📁 Structure

```
.
├── CV.typ   # Typst source file
├── CV_NguyenTanHau.pdf   # Compiled output
└── README.md
```

## ⚙️ Build

### 1. Install Typst

Download from: https://typst.app

Or use CLI (if installed):

```
typst compile cv.typ cv.pdf
```

## ✏️ Editing

* Edit content in `CV.typ`
* Recompile to update `CV_NguyenTanHau.pdf`

## 📌 Notes

* This repository includes both **source** and **output** for convenience
* `CV_NguyenTanHau.pdf` is the final version intended for sharing
